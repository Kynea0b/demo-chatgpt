# Open API

このプログラムは、chat gptのgolang実装です。

## Setting
openaiのサイトから、tokenを取得し、`.zshrc`に次のように設定してください。
https://platform.openai.com/docs/overview

```shell
export OPENAI_API_KEY=""
```

## QuickStart

次のコマンドで実行します。

```shell
go run main.go
```

実行例です。プロンプトから、質問してみましょう。

以下に、スラブに効果的な体幹トレーニングをいくつか紹介します。

1. プランク

腹筋を鍛える効果的なエクササイズで、体幹の安定性を高めます。仰向けに寝て、脚を上げ下げします。体を斜面に沿って安定させるために必要な筋力を養います。

これらの体幹トレーニングを継続的に行うことで、登りやすい姿勢を保ちつつ効率的にスラブを登ることができるようになります。

- 100gの砂糖

- 大さじ2の小麦粉
- 大さじ1のバター

作り方:

4. パイプレートにクリスプの生地を敷き詰め、リンゴの具を上にのせます。

7. 200度に予熱したオーブンで約40分間焼き、表面がきれいに焼き色が付くまで冷ましてから、お召し上がりください。

Ask a question: チョコケーキの作り方教えて
チョコケーキの作り方をご紹介します。以下は基本的なチョコケーキのレシピです。

### 材料

- 薄力粉：100g
- ココアパウダー：30g
- 砂糖：100g
- バター：70g
- 牛乳：100ml
- 卵：2個
- ベーキングパウダー：5g
- バニラエッセンス：少々
- 塩：ひとつまみ

### 作り方

1. **準備**:
   - オーブンを170℃に予熱します。
   - 型（20cmの丸型や角型）にバターを塗り、底にクッキングシートを敷いておきます。

2. **バターと砂糖を混ぜる**:
   - ボウルにバターを入れて柔らかくなるまで混ぜ、砂糖を加えてクリーム状になるまでよく混ぜます。

3. **卵を加える**:
   - 卵を1個ずつ加え、その都度しっかりと混ぜます。

4. **粉類をふるい入れる**:
   - 別のボウルで薄力粉、ココアパウダー、ベーキングパウダー、塩をふるい入れ、混ぜ合わせます。

5. **生地を合わせる**:
   - 粉類をバターと卵の混ぜたものに加え、牛乳とバニラエッセンスも加え、さっくりと混ぜ合わせます。

6. **型に流し込む**:
   - 焼き型に生地を流し込み、表面を平らに整えます。

7. **焼く**:
   - 170℃のオーブンで約30〜35分焼きます。焼きあがったら、型から外して冷まします。

8. **仕上げ**:
   - お好みで生クリームやチョコレートガナッシュをトッピングして完成です！
